# 🏥 Clinical Note Summarizer + ICD-10 Suggestion App

A biomedical NLP pipeline that takes a doctor's free-text clinical note and returns 
a concise abstractive summary and ranked ICD-10 diagnosis code suggestions — powered 
by transformer models from HuggingFace.

> Built as part of my BSc in AI with Biomedical Engineering at FAU Erlangen-Nürnberg.  
> ⚠️ For educational purposes only. Not intended for clinical decision-making.

## 🔍 What it does

Paste any free-text clinical note and get back:

- **Abstractive summary** generated by DistilBART-CNN transformer
- **Top-3 ICD-10 category suggestions** with confidence scores via zero-shot 
  classification (BART-large-MNLI)

No rule-based matching. No hardcoded keywords. Pure transformer inference.

Hypertension and high blood pressure (I10-I15)  —  93%
Stroke and cerebrovascular disease (I60-I69)    —  38%
Urinary tract infection and kidney disease       —  18%

## ⚙️ How it works

### 1. Summarisation
The input note is passed to `sshleifer/distilbart-cnn-12-6` via the HuggingFace 
`AutoModelForSeq2SeqLM` API. The model generates an abstractive summary — it writes 
new sentences rather than extracting existing ones.

### 2. ICD-10 Zero-Shot Classification
The note is passed to `facebook/bart-large-mnli` with a curated list of ICD-10 
category labels written as natural-language descriptions. The model returns confidence 
scores without any task-specific fine-tuning — demonstrating generalisation of large 
language models to the medical domain.
